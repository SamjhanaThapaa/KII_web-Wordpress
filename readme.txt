1. Open XAMPP and start the apache and mysql server
2. After you have started the servers, go to the location where you have installed xampp.
3. Open the folder xampp and go to htdocs folder
4. Paste the wordpress folder inside the htdocs folder
5. After this is done, open browser and run http://localhost/phpmyadmin/index.php
6. Create a new database called wordpress
7. open new tab and type in localhost/wordpress to open the website
